# Short Summary (Elevator Pitch)

"When Cows Fly!" is a 2D endless runner game developed in Python using the Kivy framework. Players control a flying cow, navigating an ever-scrolling landscape by tapping to ascend against gravity. The objective is to survive as long as possible by dodging various obstacles like electric wires, holes, and kites, while collecting grass to achieve a high score. The game features a shop where players can spend accumulated points on cosmetic upgrades, including new cow skins and backgrounds, with all progress and settings saved locally.
